# 博德之門：墮入阿弗納斯

**模組**：Baldur's Gate: Descent into Avernus (BGDIA)
**開始日期**：2026-04-07
**頻道 ID**：1491088654893056002
**遊戲模式**：單人 + NPC 搭檔
**當前章節**：第二章完成 → 第三章開始

## 劇情前提

神聖城市艾圖瑞爾已被拖入九層地獄第一層「阿弗納斯」。博德之門正面臨同樣的命運。來自現代世界的穿越者魯基尼，在對這個世界一無所知的情況下，被捲入了一場牽連地獄與凡間的巨大陰謀。

## 隊伍成員

- **魯基尼**（玩家角色）— 魔法騎士戰士 6 級，穿越者，變體人類
- **蒂德莉特**（NPC 搭檔）— 自然領域牧師 6 級，高等精靈
- **露露**（NPC 同伴）— 聖象（Hollyphant），薩瑞爾墮落前的夥伴，失憶中

## 當前進度

### Session 0（角色創建）✓
- 魯基尼：變體人類戰士，穿越者背景（現代世界車禍穿越）
- 蒂德莉特：高等精靈牧師，自然領域，靈感來自羅德斯島戰記
- 在博德之門城門被佐奇上尉強制徵召，任務：查三死神教

### Session 1 ✓（博德之門：街頭到陰謀）
**事件一：巴西利斯克城門**
- 目擊難民被火拳強行驅離；佐奇上尉強制徵召

**事件二：艾爾芙之歌酒館**
- 擊殺死眼海盜團（7 回合戰鬥），升至 2 級
- 塔琳娜提供三死神教浴場巢穴情報

**事件三～四：浴場地下室滲透（主線）✓**
- 清除三死神教巢穴，擊殺骨頭祭司、莫特洛克（HP 52）
- 解救俘虜（葛林、弗里達），取得靈魂冠框架
- 收服賈巴茲，獲取范撒普府邸情報

**事件五～七：范撒普府邸潛入與書房對質 ✓**
- 夜間潛入，與夫人正面對質
- 揭露薩瑞爾、艾圖瑞爾陷落真相、地獄契約全貌
- 達成暫時協議：11 天內尋找廢除原始契約的第三條路
- 升至 3 級

### Session 1 延伸 ✓（燭堡：研究與準備）
- 前往燭堡，與希薇拉·薩維卡斯合作
- 打開克雷格的地獄謎題盒，翻譯完整合約（三部分）
- 發現博德之門是薩瑞爾的第二目標
- 認識露露（聖象），決定帶她前往阿弗納斯
- 購入火焰抗性護符、收納袋
- 前往崔席格的塔樓，準備異界傳送

### Session 2–4 ✓（第二章前半：艾圖瑞爾探索與高堂大教堂）
- 異界傳送抵達阿弗納斯上空的艾圖瑞爾

- 認識費莉亞·金克斯（托姆侍祭），得知馬塔爾大公爵失蹤於大墓園

### Session 5 ✓（第二章後半：大墓園與驅邪儀式）
- 深入大墓園，與基迪恩·萊特沃德（墮落拉桑德祭司）交涉
- 獲得梅佐洛斯（蟲族傭兵）協助
- 從大墓園納骨堂救出馬塔爾（拉文加德大公爵），處於托姆之眼頭盔詛咒昏迷
- 帶回高堂大教堂，費莉亞主持驅邪儀式，蒂德莉特以抑制元素護盾保護
- 馬塔爾醒來，分享托姆的四段異象（血戰女兵、金色飛象、寶劍與堡壘、鴉人與戰車）
- 馬塔爾贈予 +1 長劍，託付尋找札瑞爾之劍的使命
- 里程碑升級：5→6 級

### Session 6 ✓（第二章收尾：準備離開艾圖瑞爾）
- 馬塔爾否決帶走費莉亞的提議（需照顧 137 名倖存者）
- 馬塔爾提供戰術簡報：東南鎖鏈路線為最佳下降點
- 隊伍完成 8 小時長休，全員滿狀態
- 補給完成（3 天份乾糧與飲水）
- **第二章完成，即將進入第三章：阿弗納斯荒野**

### Session 7 ✓（Chapter 3 — 骨節堡完成 & 出發）
- 完成瘋乖乖三個任務：米奇拔骨、巴納巴斯找牙、惡魔絞碎機修復
- 夢境儀式完成（五層夢境全通過）：彩虹天使→背叛戰場→魔鬼交易（拒絕）→荒蕪泥沼→啟示
- 露露恢復記憶：札瑞爾的劍在「血淚堡壘」
- 獲得折磨者戰車 + 靈魂幣 ×3
- 離開骨節堡，前往哈魯曼之丘

### Session 8（Chapter 3 — 哈魯曼之丘 & 路徑選擇）
- 抵達哈魯曼之丘：目睹懦夫之路——鐵樹上釘滿地獄騎士叛逃者，漲蟲循環吸血維持痛苦
- 山頂遭遇詹德・桑斯塔（精靈/吸血鬼，被釘在最大鐵樹上）——罪行：封印傳送門，斷絕札瑞爾退路
- 魯基尼主動呼喊哈魯曼，與 CR13 納祖貢展開對話（未戰鬥）
- 哈魯曼回答三個問題：納祖貢本質、為何不了結囚犯、艾圖瑞爾墜落正當性（短暫承認「人民沒有簽字」）
- 魯基尼表明「導正」意圖（非推翻），哈魯曼放行並給出忠告
- 露露承認血淚堡壘方向記錯，提出兩條線索：惡魔誕生之地（孳生樹）& 惡魔消滅之地（惡魔消滅器）
- 蒂德莉特分析後選擇前往惡魔消滅之地（Path of Devils），理由：魔鬼講秩序可談判、光柱好辨認
- 出發前往惡魔消滅器（Demon Zapper），西偏南方向
