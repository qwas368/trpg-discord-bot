# 博德之門：墮入阿弗納斯

**模組**：Baldur's Gate: Descent into Avernus (BGDIA)
**開始日期**：2026-04-07
**頻道 ID**：1491088654893056002

**當前章節**：第一章結束 → 第二章過渡中

## 劇情前提

神聖城市艾圖瑞爾已被拖入九層地獄第一層「阿弗納斯」。博德之門正面臨同樣的命運。來自現代世界的穿越者魯基尼，在對這個世界一無所知的情況下，被捲入了一場牽連地獄與凡間的巨大陰謀。

## 隊伍成員

- **魯基尼**（玩家角色）— 魔法騎士戰士 3 級，穿越者，變體人類
- **蒂德莉特**（NPC 搭檔）— 自然領域牧師 3 級，高等精靈
- **露露**（NPC 同伴）— 聖象（Hollyphant），薩瑞爾墮落前的夥伴，失憶中

- 升至 3 級
